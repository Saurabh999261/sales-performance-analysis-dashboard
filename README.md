# 📊 Sales Performance Analysis Dashboard — Power BI

## 📌 Project Overview
Built an end-to-end Business Intelligence dashboard using Power BI
on 20,000+ retail sales records to analyze sales performance and
support strategic business decisions.

## 🎯 Objective
To transform raw sales data into actionable business insights
through interactive KPI reporting and dashboard development.

## 📊 Dataset Details
- Records: 20,000+
- Dimensions: Product, Region, Time
- Period: 2014 - 2017

## 🔍 Key Insights Delivered
- YTD Sales and YoY Growth % across 4 years
- Cart segmentation and discount impact analysis
- Regional and product performance comparison

## 🛠️ Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Power Query (ETL)
- Star Schema Modeling
- Excel

## ⚙️ Key Features
| Feature | Details |
|---|---|
| Data Modeling | Star Schema with 3 dimensions |
| ETL | Power Query transformations |
| KPI Measures | 5+ DAX measures |
| Reports | YTD, YoY Growth, Segmentation |
| Impact | Reduced manual data prep by 70% |

## 📈 Project Steps
1. Data loading and ETL via Power Query
2. Star Schema modeling
3. DAX measures creation
4. KPI Dashboard development
5. Sales trend and performance analysis
6. Business insights and recommendations
